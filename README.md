## Descripción
Proyecto desarrollado para la Actividad 04 del curso de Inteligencia Artificial.

Se implementaron tres algoritmos de clustering:

- K-Means
- DBSCAN
- Spectral Clustering

Los algoritmos fueron evaluados sobre tres conjuntos de datos:

1. Iris
2. Moons
3. Circles

### Métricas externas

## Conclusiones

- K-Means presentó dificultades en problemas no lineales.
- DBSCAN mostró mejor adaptación a estructuras irregulares.
- Spectral Clustering obtuvo el mejor desempeño en el dataset Circles debido a su capacidad para modelar relaciones no lineales.
